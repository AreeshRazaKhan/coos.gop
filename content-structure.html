Website Content Structure Replacement Brief.

Page-by-page, section-by-section content map for russellcleveland.org. Every text block lists a word-limit target, every image lists render dimensions + keyword suggestions. Replace copy directly into the slots below; staying within the listed word counts keeps the layout intact.

— How to read this document

Eyebrow — small uppercase mono label above a heading. Hard cap: 5 words / 32 characters across the entire site.

Heading — display Fraunces. Word limit shown per slot. The italic accent word must be emotionally loaded, never a neutral noun.

Body / Lede — running paragraph copy. Word limit shown per slot. Body never drops below 16px on screen.

Image — render width × height (in CSS pixels at 1× density). Suggested asset is 2× of the listed dimensions for retina. Keywords drive photo selection.

CTA — button labels stay between 1–4 words.
